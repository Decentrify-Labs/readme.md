## Featured Projects

### DeMemo - Decentralized AI Memory Infrastructure

[Repository](https://github.com/Decentrify-Labs/Dememo) | [Documentation](https://github.com/Decentrify-Labs/Dememo#readme) | [Smart Contract](https://github.com/Decentrify-Labs/Dememo/blob/main/src/contracts/MemoryStorage.sol)

DeMemo is our flagship enterprise-grade decentralized AI memory layer that establishes a new paradigm for how AI systems store and retrieve conversational context. Built on Arc Network, it provides wallet-based identity, end-to-end encryption, and permanent decentralized storage through IPFS.

**Key Features:**

- **Wallet-Based Authentication** - Self-custody identity via MetaMask & Circle Wallets
- **End-to-End Encryption** - AES-256-GCM client-side encryption
- **IPFS Storage** - Permanent decentralized storage through Pinata
- **Blockchain Indexing** - Immutable on-chain memory references on Arc Network
- **AI Integration** - Contextual AI with Groq models and persistent memory
- **Microtransactions** - Cost-effective at 0.01 USDC per memory record

**Technology Stack:**
```
Frontend:    Next.js 15, React 19, TypeScript, Tailwind CSS
Blockchain:  Arc Network, ethers.js v6
Storage:     Pinata IPFS
AI:          Groq (BYOK Architecture)
Encryption:  Web Crypto API (AES-256-GCM)
Wallets:     MetaMask, Circle Developer-Controlled Wallets
```
